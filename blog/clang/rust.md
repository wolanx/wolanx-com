Cargo：Rust 的构建工具和包管理器

- cargo new hello-rust
- cargo build 可以构建项目
- cargo run 可以运行项目
- cargo test 可以测试项目
- cargo doc 可以为项目构建文档
- cargo publish 可以将库发布到 crates.io
- cargo --version

# install

```shell
# windows
https://rustup.rs/

# linux macos
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh

# update
rustup update
```
